### HOW TO SET UP:
NOTE: CMU Graphics currently on supports Python 3.9 - 3.13. If you have python 3.14, you will have to install an earlier version.

1. Install [Desktop CMU Graphics](https://academy.cs.cmu.edu/desktop) and [Python](https://www.python.org/) if needed.
2. Unzip the cmu_graphics_installer folder, and move *fnaf_at_reds.py* next to the *cmu_graphics* folder.
3. Run in your prefered method for running python programs.
4. Have fun!

### CONTROLS:
Use the mouse to interact with the game.

### GOAL:
Don't let the evil scary red ball into your office! Survive until 6:00 AM.

Watch where he is on the camera, and if he gets near your office, he could be right outside your door any moment. Flash the lights to check if he's outside your door, and shut the door to make him go away.
